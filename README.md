# DSA Practice

A collection of Data Structures and Algorithms problems solved in TypeScript.

## Structure

Problems are organized by topic, then by difficulty (`easy`, `mid`, `hard`).
Sorting algorithms are organized by algorithm type instead of difficulty.

```
dsa-practice/
├── array/
│   ├── easy/
│   ├── mid/
│   └── hard/
├── binary-search/
│   ├── easy/
│   ├── mid/
│   └── hard/
└── sorting/
    ├── bubble-sort/
    ├── insertion-sort/
    ├── selection-sort/
    └── merge-sort/
```

Each file includes a brief description of the problem along with its time and space complexity.
